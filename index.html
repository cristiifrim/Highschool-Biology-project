Caractere generale


  • Majoritatea traiesc in apa marilor, putine in ape dulci sau sunt terestre.
  • Corpul este nesegmentat si moale(de aici si numele), bogat in glande.
    El este alcatuit din corp, masa viscerala si picior musculos, diferit ca infatisare, adaptat la tipul de locomotie.
  • Simetria este bilaterala, exceptie fac gasteropodele(melcii), unde corpul si cochilia sunt rasucite in spirala.
  • Au o manta provenita din rasfrangerea tegumentului. Intre manta si corp se afla cavitatea paleala in care se gaseste aparatul respirator.
  • Mantaua secreta cochilii sau valve calcaroase.
  • Celomul se pastreaza doar in jurul inimii(pericard) si al gonadei.
  • Sistemul digestiv la unele prezinta piese de masticatie cornoase; se diferentiaza stomacul si hepatopancreasul.
  • Sistemul respirator este branhial sau pulmonar.
  • Sistemul circulator este deschis(cu exceptia cefalopodelor), format din inima situata dorsal, vase de sange si lacune.
  • Sistemul nervos are 3 perechi de ganglioni legati intre ei printre cordoane nervoase si este in legatura cu organele de simt(tentacule, ochi, etc.).
  • Au larva numita trocofora, asemanatoare cu anelidele.

  • Este clasa cu cele mai numeroase specii, multe adaptate la viata terestra.

  • Inima este de obicei, bicamerala formata dintr-un atriu si un ventricul.
  • De obicei, au un singur organ excretor, din cauza rasucirii cochiliei.
  • Inmultirea se face prin oua fara metamorfoza(la melcii terestrii) si cu metamorfoza(larva trocofora) la cei acvatici; sunt hermafrodite.
